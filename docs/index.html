PSYGIG Mobility IoT SDK¶

PSYGIG Mobility IoT SDK or psyiage is a multiplatform development kit for managing high-performance, safety-critical mobile IoT devices. The SDK provides portable, lightweight libraries and tools for provisioning, monitoring and connecting embedded devices to the cloud. The goal is to simplify the process of developing, deploying and monitoring cloud-enabled IoT edge devices.

PSYGIG Mobility IoT SDK was designed to be integrated into existing applications with minimal/no changes to the source code, along with several other features suitable for integration into embedded systems:

  • Multiplatform support

  • Third Party Integration

  • C99-standard compliant

  • Works on existing applications; no need to recompile source code

  • Export collected data into various formats (JSON, CSV, HTML)
