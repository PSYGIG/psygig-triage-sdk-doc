PSYGIG Triage SDK¶

PSYGIG Triage SDK or psyiage is a set of portable, lightweight library and tools for extending high-performance, safety-critical embedded devices with diagnostic and performance data instrumentation capabilities. The goal is to provide actionable insights in order to resolve the most critical software defects that cause system failure.

PSYGIG Triage SDK is rich with advanced instrumentation features, yet exposes a simple API for easy integration with existing software.

Application performance profiling
Software crash reporting
Log collection and monitoring
Sensor data/metric collection
System information collection
Event/alert reporting
Benchmark testing
Diagnostic testing

PSYGIG Triage SDK was designed to be integrated into existing applications with minimal/no changes to the source code, along with several other features suitable for integration into embedded systems:

  • Multiplatform support
  • Third Party Integration
  • C99-standard compliant
  • Works on existing applications; no need to recompile source code
  • Export collected data into various formats (JSON, CSV, HTML)
