Function psyiage_metric_record_values_numeric¶

  • Defined in File psyiagesdk.h

Function Documentation¶

PSYIAGE_API int psyiage_metric_record_values_numeric(psyiage_handle handle, char *names[], float values[], int nummetrics)¶

Push numeric metric values.

Record new numeric metric values into the current session.

Return

If successful, PSYIAGE_ERROR_SUCCESS. Otherwise, a non-zero error code

Parameters

  • handle: - A handle to the psyiage instance

  • names: - Array of strings identifying the metrics to push

  • values: - Array of floating point values corresponding to the metric to push

  • nummetrics: - Number of metric values to push
