PSYGIG Mobility IoT SDK¶

PSYGIG Mobility IoT SDK または psyiage は、診断およびパフォーマンスデータ計測機能を備えた、高性能で安全性が重要な組み込みデバイスを拡張するための、ポータブルで軽量のライブラリ およびツールセットです。目標は、システム障害を引き起こす最も重大なソフトウェアの欠陥を解決するために実用的な洞察を提供することです。

PSYGIG Mobility IoT SDK は高度な計装機能を豊富に備え、ソフトウェアと簡単に組み込めるためのシンプルなAPIも備わっています。

ビデオストリーミング

PSYGIG Mobility IoT SDK は、アプリケーションに統合するように設計されています、ソースコードへの変更は最小限/変更なしで、他のいくつかの機能も加えて組み込みシステムへの 統合に適した機能：

  • マルチプラットフォームサポー

  • サードパーティー導入

  • C99規格に準拠

  • アプリケーションで動作でき、 ソースコードを再コンパイルする必要はありません

  • 収集したデータをJSON, CSV, HTMLなど、様々な形式にエキスポート

ちなみに

入門 を使用すれば、すぐにアプリケーションへの計装を開始しできます。
