サードパーテイインテグレーション¶

PSYGIG Mobility IoT SDK は、サードパーティのデータプラットフォームとシームレスに導入できるため、データをストレージ/分析プラットフォームに転送するプロセスが簡単になります。

注釈

サードパーティプラットフォームのサポートは開発中です。使用プラットフォームがサポートされているかを確認する場合は 連絡 ください。
