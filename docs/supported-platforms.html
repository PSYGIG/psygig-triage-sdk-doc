Supported Platforms/Architectures¶

PSYGIG Triage SDK is designed to be integrated with embedded applications, which can come in a variety of architectures and platforms.

Platform architectures supported¶

Architecture ABI Operating System
x86-64 (x64, AMD64) x86_64 Linux / FreeBSD
ARMv5 arm Linux
ARMv6 armhf Linux
ARM64 (ARMv8) aarch64 Linux / FreeBSD

Platform architectures partially supported¶

Architecture ABI Operating System
x86 i386 Linux / FreeBSD
IA-64 ia64 Linux
PowerPC64 ppc64 Linux
MIPS mips Linux
SuperH sh4 Linux
PA-RISC hppa Linux
TILE-Gx tilegx Linux
